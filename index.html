Expressive Drum Set in 3D User Space (2020 Fall)

Froms of expression include types of input that are different from the traditional mouse and keyboard. For example, one type of expressive input could be a camera or sensor, similar to Xbox Kinect. In our project, we decided to use the front facing laptop camera as input in order to simulate playing the drums.
